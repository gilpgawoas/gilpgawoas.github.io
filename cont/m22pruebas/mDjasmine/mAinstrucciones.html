1. Este ejercicio usa la librería Jasmine para JavaScript. Puedes profundizar en este tema en la URL https://jasmine.github.io/

  2. Descarga el archivo /src/jasmine.zip y descompáctalo.

  3. Edita los archivos que sean necesarios. El código a probar se coloca en la carpeta src y las pruebas en la carpeta spec.

  4. El proyecto ya contiene la carpeta node_modules y el archivo package-lock.json, pero si quieres crearlos, debes seguir estos pasos:

    1. Si trabajas desde Windows, instala Node.js desde https://nodejs.org/en. Crea una carpeta y pásate a ella desde el símbolo del sistema. también puedes abrir la carpeta con Visual Studio Code y mostrar una terminal. Inicializa un proyecto de node con la instrucción
      npm init -y

    2. Edita el archivo packege.json e inserta en la linea 6, sin borrar ni sobreescribir nada, la siguiente línea
      "type": "module",

    3. En el mismo archivo packege.json cambia la línea que dice "test" a que diga:
      "test": "jasmine"

    4. Instala Jasmine en el proyecto con la orden:
      npm install --save-dev jasmine

    5. Inicializa Jasmine con la orden:
      npx jasmine init

  5. Ejecuta las pruebas con la orden:
    npm run test
