1. Este ejemplo requiere 2 proyectos

  2. Revisa el proyecto en phpmensajes en Replit con la URl https://replit.com/@GilbertoPacheco/phpmensajes?v=1

  3. Revisa el proyecto htmlmensajes en Replit con la URl https://replit.com/@GilbertoPacheco/htmlmensajes?v=1

  4. htmlmensajes puede correr simultáneamente en varios navegadores y computadoras, todos interactuando con la misma instancia de phpmensajes, obteniendo un chat muy simple.

  5. Usa o crea una cuenta de Google.

  6. Crea una cuenta de Replit usando la cuenta de Google.

  7. Crea el proyecto phpmensajes con la categoría PHP CLI en Replit y edita o sube los archivos de este proyecto.

  8. Para el contenido de la carpeta vendor y los archivos composer tienes 2 opciones:

    • Descargar el siguiente archivo zip: composer.zip.

    • Crear una carpeta, usar un shell como CMD, sh o PowerShell en esa carpeta, tener PHP y composer para PHP instalados en tu computadora y usar la orden:

      composer require cboden/ratchet

  9. Depura el proyecto.

  10. Crea la página de spotlight del proyecto.

  11. Crea el proyecto htmlmensajes con la categoría HTML, CSS, JS en Replit y edita o sube los archivos de este proyecto.

  12. Depura el proyecto.

  13. Crea la página de spotlight del proyecto.
