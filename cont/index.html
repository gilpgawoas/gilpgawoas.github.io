por Gilberto Pacheco Gallegos

  • Este curso trata sobre páginas web que se comunican, sin recargarse, con un tipo de software conocido como servicio, alojado en servidores.

  • Otros tipos de app, como apps móviles y de escritorio también se pueden conectar a un servicio.

  • Este curso también se muestra la construcción de mashups, técnica que consiste en agregar el contenido de otros sitios.

  • También se muestran los fundamentos para construir e-commerce.

  • Los chats con websockets se han sustituido por un chat con MQTT, que es más sencillo.

  • Utiliza el botón para mostrar el menú de navegación, o el enlace skip_next para avanzar a las siguietes paginas. La página Instrucciones de navegación te muestra otras opciones para desplazarte por el sitio.
