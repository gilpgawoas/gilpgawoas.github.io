B. Instrucciones

  1. Este ejemplo requiere 2 proyectos

  2. Revisa el proyecto en phpmensajes en Replit con la URl https://replit.com/@GilbertoPacheco/phpmensajes?v=1

  3. Revisa el proyecto htmlmensajes en Replit con la URl https://replit.com/@GilbertoPacheco/htmlmensajes?v=1

  4. Descarga phpmensajes.

  5. htmlmensajes puede correr simultáneamente en varios navegadores y computadoras, todos interactuando con la misma instancia de phpmensajes, obteniendo un chat muy simple.

  6. Usa o crea una cuenta de Google.

  7. Crea una cuenta de Replit usando la cuenta de Google.

  8. Crea el proyecto phpmensajes con la categoría PHP CLI en Replit y edita o sube los archivos de este proyecto.

  9. Para el contenido de la carpeta vendor y los archivos composer tienes 2 opciones:

    • Descargar el siguiente archivo zip: composer.zip.

    • Crear una carpeta, usar un shell como CMD, sh o PowerShell en esa carpeta, tener PHP y composer para PHP instalados en tu computadora y usar la orden:

      composer require cboden/ratchet

  10. Depura el proyecto.

  11. Crea la cover page o página de spotlight del proyecto.

  12. Crea el proyecto htmlmensajes con la categoría HTML, CSS, JS en Replit y edita o sube los archivos de este proyecto.

  13. Depura el proyecto.

  14. Crea la cover page o página de spotlight del proyecto.

E. Eventos.php

<?php

use
 Ratchet\MessageComponentInterface;
use Ratchet\ConnectionInterface;

class Eventos implements
 MessageComponentInterface
{
 protected $conexiones;

 public function __construct()
 {
  // Crea un arreglo de objetos.
  $this->conexiones =
   new \SplObjectStorage;
 }

 public function onOpen(
  ConnectionInterface $con
 ) {
  // Guarda la conexión.
  $this->conexiones->attach($con);
  echo "Conectado: " .
   $con->resourceId . "\n";
 }

 public function onMessage(
  ConnectionInterface $origen,
  $msg
 ) {
  echo "Recibido: $msg\n";
  // Reenvía el mensaje a todos.
  foreach ($this->conexiones
   as $con) {
   $con->send($msg);
  }
 }

 public function onClose(
  ConnectionInterface $con
 ) {
  $this->conexiones->detach($con);
  echo "Desconectado: " .
   $con->resourceId . "\n";
 }

 public function onError(
  ConnectionInterface $con,
  \Exception $e
 ) {
  echo "Error: " .
   $e->getMessage() . "\n";
  $con->close();
 }
}

F. main.php

<?php
require __DIR__ .
 '/vendor/autoload.php';
require_once "./Eventos.php";

use Ratchet\Server\IoServer;
use Ratchet\Http\HttpServer;
use Ratchet\WebSocket\WsServer;

$server = IoServer::factory(
 new HttpServer(
  new WsServer(
   new Eventos($valor)
  )
 ),
 80
);

$server->run();

<!DOCTYPE html>
<html>

<head>
 <meta charset="UTF-8">
 <meta name="viewport" content="width=device-width">
 <title>Mensajes</title>
</head>

<body>
 <form id="forma" action="javascript:envia()">
  <h1>Mensajes</h1>
  <p>
   <label>
    Alias
    <input name="alias">
   </label>
  </p>
  <p>
   <label>
    Mensaje
    <input name="mensaje">
   </label>
  </p>
  <p>
   <button>Enviar</button>
  </p>
 </form>
 <pre id="pre"></pre>
 <script>
  try {
   /* Cambia la url por la de tu
    * servicio. */
   const ws = new WebSocket(
    "wss://phpmensajes.gilbertopacheco.repl.co")
   ws.onopen = wsAbierto
   ws.onmessage = recibeMensaje
   ws.onerror = error
   ws.onclose = cerrado

   /** @param {MessageEvent} evt */
   function recibeMensaje(evt) {
    pre.textContent +=
     evt.data + "\n\n"
   }

   function envia() {
    const msg =
     `${forma.alias.value}
${forma.mensaje.value}`
    ws.send(msg)
   }

   function wsAbierto() {
    console.log('Conectado')
   }

   function error(evt) {
    console.error(evt)
    alert("Error en la conexión.")
   }

   function cerrado(evt) {
    console.log(evt)
    alert("Conexión terminada.")
   }
  } catch (e) {
   console.log(e)
   alert(e.message)
  }
 </script>
</body>

</html>
